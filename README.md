# Product Management CRUD – ASP.NET Core MVC

A simple Product Management application built using ASP.NET Core MVC and Entity Framework Core.

## Features
- Create, Read, Update, Delete Products
- Server-side validation
- Clean MVC architecture
- SQL Server database

## Technologies Used
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Razor Views

## Getting Started

### Prerequisites
- .NET 6 or later
- SQL Server
